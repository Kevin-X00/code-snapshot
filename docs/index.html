code-snapshot

Merge your entire codebase into one text block for AI agents. Stop copy-pasting files manually.

Install globally:

npm install -g @kevinxyz/code-snapshot

Then run anywhere:

snap ./src | claude ⚡ Works with Claude Code, ChatGPT, Codex, Cursor, Copilot — any AI agent.
⚡

One Command

snap ./src — walks your directory, respects .gitignore, wraps files with clear markers.

🎯

Git-Aware

--git-only includes only changed files. --minify strips comments to save tokens.

🧩

Flexible Output

Pipe to AI, write to file, copy to clipboard. --format xml, --json, or default markers.

📊

Token Aware

Every output shows estimated token count. --tokens flag, --top-files to spot bloat.

💬

Interactive Mode

snap . -i — pick files, choose format, decide output. Perfect when you need control.

🔌

Universal

Works with any AI agent. snap . | claude, pipe to ChatGPT, or paste into Cursor.

Quick Examples

snap . | claude
Snapshot your whole project and pipe into Claude Code
snap . --git-only -o diff.txt
Only files changed since last commit
snap . -i --tokens
Interactive file picker with token estimation
snap ./src --minify --exclude "*.test.js"
Minify code, skip test files
snap . --json -o codebase.json
Machine-readable JSON output
